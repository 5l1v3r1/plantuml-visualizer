## How to run this plugin

1. `yarn` to install dependencies
1. `yarn build`
1. Open Chrome
1. Open [chrome://extensions](chrome://extensions)
1. Enable `Developer Mode`
1. Click `Load Unpacked` and open this directory (`plantuml-visualizer`)
